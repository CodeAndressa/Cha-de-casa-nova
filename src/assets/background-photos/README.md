# Fotos de fundo

Coloque aqui as fotos que devem aparecer pequenas e se movimentando no fundo da home.

Formatos aceitos: `.jpg`, `.jpeg`, `.png`, `.webp` e `.avif`.

Exemplos de nomes:

- `sala.jpg`
- `cozinha.webp`
- `casal-01.png`

Depois de adicionar as imagens, fa?a commit e push. O site carrega automaticamente tudo que estiver nesta pasta.
